# michael-harris-sweng / dotfiles

## Overview
Contains configuration files for various unix utilities, to simplify future installations.

## File Structure
```YAML
dotfiles/
├── git/
│   └── .gitconfig # git configuration file.
└── zsh/
    └── .zshrc     # zsh (oh-my-zsh) configuration file.
```
